## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. For production build:

   ```bash
   npm run build
   ```

4. To run tests:
   ```bash
   npm test
   ```
